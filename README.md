# Pearl Landing Timer

A client-side Fabric mod that shows a configurable HUD countdown for your latest thrown ender pearl.

## Features

- Predicts when your latest ender pearl will land.
- Displays the countdown in seconds or Minecraft ticks.
- Includes Mod Menu and Cloth Config settings.
- Lets you reposition the HUD preview in game.

## Requirements

- Minecraft 1.21.x through 1.21.11
- Fabric Loader 0.19.2 or newer
- Fabric API
- Cloth Config
- Mod Menu
