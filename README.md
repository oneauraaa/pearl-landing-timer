# Pearl Landing Timer

Pearl Landing Timer is a mod that shows a configurable HUD countdown for your latest thrown ender pearl. It helps you see when your pearl is predicted to land, with options for seconds or Minecraft ticks.

## Features

| Feature | Description |
| :--- | :--- |
| Pearl landing countdown | Shows a timer for your latest thrown ender pearl. |
| Timer display modes | Display the countdown in seconds or Minecraft ticks. |
| Configurable HUD | Move the timer HUD to your preferred screen position. |
| Optional visuals | Toggle the pearl icon and background on or off. |
| Mod Menu support | Configure the mod from the Mod Menu settings screen. |

## Supported Platforms & Versions

| Platform | Supported Versions |
| :--- | :--- |
| Fabric | 1.21.x |
